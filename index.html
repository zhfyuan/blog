Markdown

引子

Markdown is a way to write content for the web.
鉴于Markdown使用的越来越多，不光hexo写blog用到，jupyter notebook也用到，所以这里总结一下基本用法。
更多内容

常用符号

  • *
    斜体，星号夹住文本，就会变成斜体。_下划线也有类似功能。

  • **
    粗体，两个星号夹住文本，就会变成粗体。__两个下划线类似效果。
    三个星号可以同时斜体加粗 比如，**_似乎在hexo中不能成功。

  • ～～
    删除一段文字

  • #
    #代表标题，如果有两个就是二级标题，共有六级。注意#要打头且和标题正文之间至少留一个空格.

    标题二

    标题四

    标题六

  • inline link
    隐式链接，中括号内放置显示链接提示，小括号内放置真正链接网址。

  • reference link
    显式链接，好处是作为参考链接，多处引用的话，如果修改只需要改一次，现在用google很不方便，但是用baidu搜学术论文很不方便，而google在这方面就很有优势。

2008年的时候，诸事不顺，自己写过那么几首打油诗

蛋壳

你走了
天塌了
我哭了
良久
才发现
别有洞天

受伤

受伤
经过伤心事
怎能不受伤
受伤又能怎么样
还不是受伤

我中毒啦

我中毒啦中毒啦
我都不知道怎么去杀

挨个的格式化啊

你说是不是早晚得回老家

思念的滋味

你知不知道
思念一个人的滋味
就像白砂糖遇见了水

你知不知道
思念一个人的滋味
就像阴天里一杯苦咖啡

思念一个人的滋味
就像棉铃虫喝了敌敌畏

你知不知道

就像蜻蜓点了一下水

你知不知道
思念一个人的滋味

然后慢慢
一滴一滴
化成热泪……

那个你，那个我

记得那一天，
我们偶相见，
上海地铁边，
夏末的傍晚；

如果有一天，
你我又见面，
当时可记得，
你是那个你，我是那个我。。。

粗糙的美

早起，阴雨。
来上海已三年挂零，却至今不能喜欢这半死不活的天气。习惯性的阴天，习惯性的绵绵细雨，总给人一种尿不净的感觉。相比起来，我更怀念北方的原野，这个时候，家乡应该满是荒凉、肃杀的气氛吧，这种大自然的原始冲动也许更能拉近我和大地之间的距离。收获的果实也应该是沉甸甸的，晴天总是唱主角的，下雨也是豆大的雨滴铺将下来，给人一种痛快淋漓的快感。慢慢地，我发现，我越来越喜欢粗线条的美，而这是上海所不能给我的。
上海似乎更易让人产生一种小资情调，混着说不清道不明的淡淡的忧郁和轻愁。看着天气，想想自己，萌出，一丝诗意。
衣衫厚薄薄复厚，
昔日同学早出走。
孤身对影自言语，
不觉已是江南秋。

无题

人生失意十八九，
自在潇洒古少有。
惟有曹州袁层次，
刀尖上面舞着走。
窗外阳光明媚，一切都在好转，写首诗见证自己此刻的心情。
①曹州：袁层次老家，今山东菏泽。
②层次：作者飞，字层次。很久不用了。

2008年期间 by zhfyuan
