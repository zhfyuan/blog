我家是湖北的

周末，天大好。
一家人去欢乐谷外绿地游玩。
太阳强烈，水波温柔。
帐篷搭好后，外甥女阳阳和旁边一个年轻妈妈带的小朋友聊在一起。
：你叫什么名字？
：畅畅（音译），你呢？
：阳阳，我读四年级了。
：在哪个学校呢？
：我不在这里上学，我家是湖北的。
畅畅，畅畅，来，过来，来这边来，我们把垫子往这边挪一点，年轻妈妈喊他的小孩。
过了大概不到十分钟，她们终于还是走了。
我想过去告诉她们，外甥女是湖北人，但是年前很早就来上海了，不用担心的。
但我终究什么也没说。

nohup&screen

hup是hang up的缩写，意思是挂断。nohup就是不要挂断。

nohup命令，使得程序忽略SIGHUP信号(当前控制终端关闭时发送到进程的信号)，也就是说，虽然终端关闭了，该命令还可以照常执行。
如果想把它放在后台运行，命令最后加 & ，这样就可以在前台输入其他命令而不受干扰。

& 放到后台运行，暂时看不到，但是还在跑。

比如我在服务器上启动一个notebook服务，命令如下，日志记录到log文件中，放在后台执行。

通过 ps -ef，或者 ps aux 或者 ps -a 都可以查看当前运行的进程。

通过grep筛选，grep -v grep是为了剔除grep自身，awk用来提取进程号（进程ID）。

通过kill命令可以杀死进程。

强制杀死。

如果开始没有加 &，运行中想把它放到后台，使用快捷键Ctrl + Z，这样进程就暂停在后台。
如果接着想恢复到前台，则使用fg命令恢复到前台执行。如果想在后台执行，则用bg命令。
如果有多个进程，则fg和bg需要加参数（jobs可以查看）
总结：fg和bg都是将停止的进程激活，一个是从后台拿到前台，一个是从前台拿到后台。

举个例子，用vim更在编辑一些东西，想调用shell命令查一些东西，可以ctrl+z，执行完后fg恢复vim。

虽然nohup很容易使用，但还是比较“简陋”的，对于简单的命令能够应付过来，对于复杂的需要人机交互的任务就麻烦了。
其实我们可以使用一个更为强大的实用程序screen。流行的Linux发行版（例如Red Hat）通常会自带screen实用程序，ubuntu没有的话

补充一个命令 tee， 不仅输入到文件，还同时显示在屏幕上. 如果用 > 则只会到文件，屏幕就不显示了。

2>&1 是数据流双定向，即无论正常和错误的都输出到这里。

balance

平衡，阴阳之道。
如何知道自己在处事为人中有没有很好的掌握好平衡点呢？
有一个简单的办法，对比对人对己。

比如，如果对别人做的某件事你很鄙视，然后对自己做这件事又很坦然，这就是平衡点没掌握好。
比如别人公交车上偷看美女，你会觉得很没品，而你自己又回不自觉偷瞄，这就说明你很刻薄。

再或者，特别鄙视巴结人，看不起巴结别人的人，而自己又特别渴望别人巴结自己，这也是没掌握好平衡点。
多中和一下，有好处。

再或者，对于比自己年长几岁的人，喜欢尊称别人老师，对于比自己小的人，又不愿别人叫自己老师。

当前并不是一定要中间位置才最好，正因为并不总是中间位置，才造就个性的人。

家乡的话

我一九八二年生，十九岁时也就是二零零一年上大学，那是头回离开家乡，到现在又有了十九个年头。差不多在外面的时间要超过家乡的时间了。
年龄越大反而越容易想起小时候的事，怀念庄头的几颗大杨树，怀念家后的那条小河，怀念小学门口的窄狭土路，怀念初中加班放学后顶着星星经过庄稼地的少年。

不像现在家家户户的高门大院，那时候都是半人高的土墙，小伙伴们玩藏盲游戏可以轻松地从一个胡同翻进另一个胡同，大门一般都敞着，甚至很多都没有大门。一群打着补丁的孩子玩得那样开心。

可是现在，一切都不见了，我的家乡，除了慢慢老去的亲人，除了在地图上不变的坐标，到底还剩下了什么。

我思来想去，什么都变了，但是喜欢吃的口味没变，说话的口音还是那味。
今天不谈喜欢吃的棒子面糊涂，也不讲皮杂油茶胡辣汤，单单说说我的家乡话。
我总结了家乡话的几个特点，记录如下：

  • 倒装
    以前网上有个段子，叫山东人说话爱倒装，有人还不承认，山东人也不全都这样啊其实。
    我想说的是，我们那里更进一步，不但句子倒装，连单个词语都倒装。
    比如，会把喷嚏说成嚏喷，把狭窄说成窄狭。
    又打嚏喷，得了狗头金啦你；
    你看这个地方多窄狭，腿都蹚不过去。

  • 复古
    家乡人好古风，狭义热肠。说话也能体现，很多话用的都是古语。
    比如，害怕说成怯，我才不怯他；碰说成遭，遭遇的遭。你遭遭我试试。
    以前中学一篇课文，蒲松龄的《狼》，身已半入，止露尻尾。
    这个尻字非常复古，普通话几乎很少用，但家乡话好像用的不少。

  • 数量词
    在数字上家乡人有独特的表达方式。
    比如普通话： 你妈去哪儿啦？ 可能去邻居家聊天了。
    家乡话：恁妈弄啥去了？ 八成去串门了。
    点评：可能或者大概，表示不是很确定，但没有说出多少可能性，八成就是80%可能，非常具体。
    另外，也有这么说的，惯性去串门了。
    惯性按照物理学里惯性定律解释，就是按照趋势判断，她去串门了。
    八成，惯性，数学、物理都有了。
    串门，把两个门串起来，就是去走动了。
    再比如，一勾（1/3，田地里三勾为一趟，一勾就是1/3）、一个半劲（你看他能嘞一个半劲，1.5倍高能）、十分九力（9/10）…

  • 北方
    家乡在长江以北，除了以前被黑龙江的同学说成是南方人，其实地地道道北方人啦。语言里也有很多烙印。
    说一个人以后能有出息，他以后能吃大馍嘞。说一个人不中用，那就是个白搭馍。
    南方就是，我知道自己能吃几碗干饭。
    北方和南方，馒头和米饭。
    家乡说吃米饭，说成大米干饭，小时候用棒子、麦换大米，吃一顿就是改善伙食了。

  • 两字词
    有的两个字，本身组成一个词语，现在还很流行。但是拆开也能用。
    比如 沉重，单独拆开，普通话会说，这个东西很重，家乡话说，这个东西很沉。
    再有 肥胖，单独拆开，普通话说，这个人很胖，家乡话说，你看那个孩的长多肥。

    如果在我的家乡，金三胖估计会被叫成三老肥。

  • 形象
    有些词非常形象，比如 普通话说 他经常迟到，他经常干龌龊的事。
    家乡话，他靠住干腌臜事。靠住，画面感一下就支棱起来了。
    再有，他上课经常发困，没精神。他上课靠住栽嘴。
    普通话人会栽树、会栽跟头，而我的家乡人会栽嘴。

  • 他特别喜欢攀比。他血好咬磨嘴。
  • 还有上面提到的串门，也非常形象。

  • 独有
    还有一些意思是独有的。
    比如，他血好挖人，你看我手上好几个口子。
    血，是特别的意思，他血能，谁都不放在眼里。
    他血怪，没谁愿意跟他玩。
    他血野，不好寻媳妇。
    挖，挠的意思，脸挖冒血了。
    怪，调皮捣蛋。
    野，个头矮。

  • 男女尊卑
    不得不承认，家乡有男尊女卑的传统，以前没有男孩叫绝户头。
    评价一个人能或者不能，不用NB也不用傻逼。而用屌，很牛就是很屌，很傻就是傻屌。
    你看他多屌能，尾（yi）巴都杠天上去了。
    别跟他一样，他是个大傻屌。
    当然现在好了很多，因为很多人开始骂傻逼了：）。

补充一句，普通话里从来不会把某两个字眼放在一起用。
瞎屌逼能。

结语

不知道若干年后，这些话还有人说吗？

Markdown

引子

Markdown is a way to write content for the web.
鉴于Markdown使用的越来越多，不光hexo写blog用到，jupyter notebook也用到，所以这里总结一下基本用法。
更多内容

常用符号

  • *
    斜体，星号夹住文本，就会变成斜体。_下划线也有类似功能。

  • **
    粗体，两个星号夹住文本，就会变成粗体。__两个下划线类似效果。
    三个星号可以同时斜体加粗 比如，**_似乎在hexo中不能成功。

  • ～～
    双波浪号夹住就会删除一段文字

  • #
    #代表标题，如果有两个就是二级标题，共有六级。注意#要打头且和标题正文之间至少留一个空格.

    标题二

    标题四

    标题六

  • inline link
    隐式链接，中括号内放置显示链接提示，小括号内放置真正链接网址。

  • link2image
    和文本链接不同的是多了个感叹号。

    本地图片链接方式，参见test-link.md

  • 引用

    在文字前面加上大于号，用于引用名人名言，或者老话说，或者俚语方言等等。

  • *

*代表列表，-类似功能。

  • 三星号
    三个星号代表分割线

[^_^] 注释
注意，注释的内容要另起一行，Tab空格。

  • 代码
    print('I Love U !')

  • 其他
    不属于markdown范畴，但是很有用。
